# AI Command Center

A private, single-user Personal AI Operating System — an executive assistant, project manager, research analyst, and strategic advisor in one app. See `docs/architecture.md` for the full architecture, database schema, and phased roadmap (mirrors the approved project plan).

## Stack

- Next.js 15 (App Router) + TypeScript + Tailwind CSS v4 + shadcn/ui
- Firebase (Auth, Firestore, Storage)
- Vercel AI SDK + AI Gateway (model routing/fallback across providers)
- Deployment: Vercel

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
